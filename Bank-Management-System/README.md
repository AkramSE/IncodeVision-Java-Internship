# 🏦 Bank Account Management System

A robust, console-based banking application developed in **Java** that follows industry-standard coding practices and focuses on secure data handling.

## 🚀 Key Features
* **Secure Transactions:** Supports Deposit, Withdrawal, and Balance inquiry with real-time updates.
* **Smart Data Retrieval:** Implements `HashMap` to ensure lightning-fast **O(1) time complexity** for accessing account details.
* **Input Validation:** Uses strict **Regular Expressions (Regex)** to ensure names and account numbers are in the correct format, preventing invalid data entry.
* **Professional Formatting:** Utilizes `String.format()` to display currency in a clean, readable way (e.g., Rs. 50,000.00).
* **Exception Handling:** Integrated with comprehensive `try-catch` blocks to handle runtime exceptions and invalid inputs gracefully.

## 🛠️ Technical Stack
* **Language:** Java (JDK 17+)
* **Data Structures:** HashMap, Collections Framework
* **Concepts:** Object-Oriented Programming (OOP), Exception Handling, Regex
