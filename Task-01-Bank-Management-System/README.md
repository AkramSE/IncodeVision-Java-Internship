# 🏦 Bank Account Management System (Task 01)

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=java&logoColor=white)
![Status](https://img.shields.io/badge/Status-Completed-success?style=flat)

Yeh project ek secure console-based banking application hai jo core banking operations ko simulate karta hai. Iska main focus efficiency aur user experience (UX) par hai.

## 🚀 Key Features
* **Efficient Lookups:** `HashMap` use kiya gaya hai taake account retrieval fast (O(1)) ho.
* **Smart Validation:** Regex ke zariye ghalat entries (jaise names mein numbers) ko roka gaya hai.
* **Professional UI:** Balance ko proper formatting (`Rs. 100,000.00`) mein dikhaya gaya hai taake `1.0E8` jaisa output na aaye.
* **Fail-Safe:** Program crash nahi hota kyunke har input ko `try-catch` se handle kiya gaya hai.

## 🛠️ Concepts Applied
* **Data Structures:** HashMap (DSA).
* **OOP:** Encapsulation & Abstraction.
* **Java Utilities:** Scanner, String Formatting, Regular Expressions.
